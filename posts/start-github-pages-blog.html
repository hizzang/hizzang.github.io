GitHub Pages 블로그 시작하기

이 블로그는 hizzang/hizzang.github.io 리포지토리를 이용해 운영됩니다. GitHub Pages는 정적 HTML, CSS, JavaScript 파일을 무료로 호스팅할 수 있어 개인 블로그에 적합합니다.

운영 방식

  • 글은 posts/ 디렉터리에 HTML 또는 Markdown 변환 결과로 추가합니다.
  • 스타일은 assets/css/style.css에서 관리합니다.
  • master 브랜치에 push하면 GitHub Pages에서 사이트를 제공합니다.

다음 단계

이후 Astro, Hugo, Jekyll 같은 정적 사이트 생성기를 붙이면 글 목록, 태그, RSS, 검색 등을 자동화할 수 있습니다. 초기에는 단순 HTML 구조로 시작하고, 글이 많아지면 정적 사이트 생성기로 전환하는 방식이 관리하기 쉽습니다.

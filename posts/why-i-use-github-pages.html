게임 플랫폼 엔지니어가 GitHub Pages를 선택한 이유

개인 블로그는 새로운 기술을 정리하고 미래의 나 자신을 위한 문서를 만드는 공간이다. 특히 인프라와 플랫폼을 다루는 엔지니어에게는 문제 해결 과정을 기록하는 것이 가장 큰 자산이 된다.

왜 GitHub Pages인가?

  • 무료 호스팅
  • Git 기반 버전 관리
  • 자동 HTTPS
  • Markdown 기반 확장 가능

앞으로 다룰 주제

  • 게임 플랫폼 아키텍처
  • CDN 및 글로벌 서비스 운영
  • Keycloak/SSO 구축
  • DevOps와 자동화
  • Electron 런처 개발
좋은 문서는 미래의 자신에게 보내는 가장 가치 있는 편지다.

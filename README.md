# hizzang blog

GitHub Pages로 운영하는 개인 기술 블로그입니다.

## URL

- https://hizzang.github.io

## 구조

```text
.
├── index.html
├── about.html
├── posts/
│   └── start-github-pages-blog.html
└── assets/
    └── css/
        └── style.css
```

## 글 추가 방법

1. `posts/` 디렉터리에 새 HTML 파일을 추가합니다.
2. `index.html`의 최근 글 목록에 링크를 추가합니다.
3. `master` 브랜치에 push하면 GitHub Pages에 반영됩니다.

## 향후 개선 후보

- Astro 또는 Hugo 도입
- Markdown 기반 글 작성
- RSS 피드
- 태그/카테고리
- Giscus 댓글
- Google Analytics 또는 Plausible 통계
